# X (Twitter) Clone

A responsive frontend clone of X (formerly Twitter) built using HTML and Tailwind CSS. This project focuses on recreating the UI and layout of the platform with a clean and modern design.

## 🚀 Features
- Responsive layout (mobile + desktop)
- Sidebar navigation with icons
- Post creation UI ("What is happening?")
- Feed with posts and interactions (like, comment, share)
- Trending section and "Who to follow"
- Sticky header and sidebar

## 🛠️ Tech Stack
- HTML5
- Tailwind CSS
- Google Material Icons

## 📂 Project Structure
- `index.html` – Main UI structure :contentReference[oaicite:0]{index=0}
- `css/output.css` – Compiled Tailwind styles
- `tailwind.config.js` – Tailwind configuration

## 🎯 Purpose
This project was built to practice:
- Tailwind CSS utility-first styling
- Responsive UI design
- Real-world frontend layout replication

## ⚠️ Note
This is a frontend-only clone and does not include backend functionality or real-time features.
